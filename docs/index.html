backup-mysql

Basic tool to backup mysql servers.

Installation

pip install --upgrade backup-mysql

Setup

Create a /etc/backups/backups.yaml file with content similar to:

backups:
      server:
        connection:
          type:                   mysql
          host:                   localhost
          username:               backup
          password:               password
        options:
          encryption:             secret
        s3:
          bucket:                 acme-backup-databases
          path:                   live/mysql/<suffix>

Now run it with

backup-mysql server
